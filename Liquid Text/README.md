🌊 **Liquid Text Animation Project** 🌊

Welcome to the mesmerizing world of Liquid Text, where text comes to life with stunning animations and a beautifully designed UI! This front-end project is your gateway to exploring the world of liquid-like transitions for text. Get ready to be captivated by an attractive and eye-catching user interface!

🚀 **Project Purpose and Description** 🚀

Liquid Text is a front-end project designed to showcase the magic of text animation. With a responsive and visually appealing user interface, this project aims to make your website's text pop and engage your audience. Whether you're a web developer looking to spice up your website or a design enthusiast seeking inspiration, Liquid Text is here to impress.

🛠️ **Tech Stack** 🛠️

- HTML
- CSS
- SCSS
- Bootstrap 5
